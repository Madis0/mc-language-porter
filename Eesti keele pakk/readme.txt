This folder only contains the base and extra files (extra-et_EE.lang) for the Estonian language pack for Minecraft.
The actual language file must be generated with the scripts designed for it.

---

See kaust sisaldab vaid baas- ja lisafaile (extra-et_EE.lang) "Eesti keele pakk Minecraftile" tarbeks.
Tegelik keelefail tuleb genereerida selleks mõeldud skriptide abil.
